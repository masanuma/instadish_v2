# 🤖 AI画像処理機能の設定

InstaDishでOpenAI APIを使用したAI画像処理を利用するための設定方法です。

## 📋 必要な準備

### 1. OpenAI APIキーの取得
1. [OpenAI Platform](https://platform.openai.com)にアクセス
2. アカウント作成・ログイン
3. [API Keys](https://platform.openai.com/api-keys)でAPIキーを作成
4. 作成されたAPIキーをコピー

### 2. 環境変数の設定
プロジェクトのルートフォルダに `.env` ファイルを作成し、以下を記述：

```env
# OpenAI API Configuration
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# アプリケーション設定
DEBUG=True
```

### 3. アプリの再起動
Streamlitアプリを再起動して設定を反映させてください。

## 🎨 AI処理の特徴

### ✅ 実行される処理
- **色調調整**: 業種・ターゲット層に最適な色合いに調整
- **明度・コントラスト**: プロ品質の光量調整
- **彩度調整**: 食材の魅力を最大化
- **温度調整**: 暖色・寒色の自然な調整

### ❌ 厳格な制限事項
- **形状変更禁止**: 料理や器具の形は一切変更しません
- **オブジェクト追加/削除禁止**: 写真にあるものをそのまま保持
- **構図変更禁止**: 元の構図とサイズを維持
- **テキスト変更禁止**: ラベル等の文字は変更しません

## 💡 使用上の注意

### API料金について
- DALL-E 2/3は有料サービスです
- 1回の処理で約$0.02-0.08の料金が発生
- 詳細は[OpenAI Pricing](https://openai.com/pricing)を確認

### 処理時間
- AI処理には30秒-2分程度かかる場合があります
- 失敗時は自動的に従来処理にフォールバック

### 品質について
- OpenAI APIによる高品質な色調補正
- 業種・ターゲット層の特性を考慮した最適化
- 従来処理より自然で魅力的な仕上がり

## 🔄 フォールバック機能

AI処理が失敗した場合、自動的に以下が実行されます：
1. 従来の画像処理技術による補正
2. 品質優先の画像保存
3. 業種別の色調調整

これにより、確実に画像処理が完了します。
