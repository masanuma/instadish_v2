## 技術的な実装詳細

### OpenAI API仕様
- **モデル**: DALL-E 2（画像編集用）
- **ファイル形式**: PNGのみ対応
- **ファイルサイズ**: 4MB以下
- **レスポンス形式**: URLまたはbase64

### 画像処理フロー
1. アップロードされた画像をPNG形式に変換
2. 必要に応じてマスクを生成
3. OpenAI APIに画像とマスクを送信
4. プロンプトに基づいて画像を編集
5. 結果を取得して表示

### マスク生成オプション
- **自動マスク**: 背景除去、オブジェクト検出
- **手動マスク**: Canvasベースの描画機能
- **プリセットマスク**: 業種別の標準マスク
